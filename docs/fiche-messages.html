SAYTOU — Fiche d’utilisation — Rubrique Messages

Objectif: envoyer et recevoir des messages entre les comptes autorisés (Comité Pédagogique, Sous-Localité, Sections, Cellules/Commissions).

1) Accès à la rubrique

Dans le menu à gauche, cliquez sur Messages. Vous arrivez sur 3 onglets: Boîte de réception, Nouveau message, Envoyés.

Remarque: si votre rôle n’a pas le droit d’envoyer, l’onglet “Nouveau message” peut ne pas apparaître.

2) Rôles et qui peut écrire à qui

Rôle (expéditeur) Peut envoyer à… Cas d’usage
COMITE_PEDAGOGIQUE SOUS_LOCALITE_ADMIN SECTION_USER Informer / demander des rapports aux sous-localités et sections.
SOUS_LOCALITE_ADMIN (selon configuration actuelle) Réception + réponse via Messages si l’onglet “Nouveau message” est disponible. Réception des consignes et suivi.
SECTION_USER COMITE_PEDAGOGIQUE SOUS_LOCALITE_ADMIN Répondre aux messages reçus, remonter une information.
ORG_UNIT_RESP (Cellule/Commission) SECTION_USER SOUS_LOCALITE_ADMIN COMITE_PEDAGOGIQUE Communication opérationnelle avec les sections et l’administration.
Important: La liste des destinataires est automatiquement filtrée par la même localité (même zone d’organisation), selon les règles d’accès.

3) Lire les messages (Boîte de réception)

Voir la liste

  • Onglet Boîte de réception.
  • Les messages non lus apparaissent en surbrillance.
  • Le compteur “non lu(s)” s’affiche en haut.

Marquer comme lu

  • Cliquer sur un message l’ouvre et le marque comme lu.
  • Vous pouvez aussi utiliser Tout marquer comme lu.

4) Répondre à un message

Ouvrez un message reçu puis cliquez sur Répondre. L’application prépare un nouveau message avec:

  • Le destinataire = l’expéditeur du message
  • Le sujet = Re: ...

Ensuite, saisissez votre texte puis envoyez.

5) Envoyer un nouveau message

Étapes

  1. Aller dans l’onglet Nouveau message.
  2. Choisir les destinataires (case à cocher).
  3. Utiliser les filtres de destinataires: Tous, Sections, Sous-Localités, C. Pédagogique (selon disponibilité).
  4. Renseigner Sujet et Message.
  5. Cliquer sur Envoyer.

Astuce: utilisez “Tout sélect.” / “Tout désélect.” pour gagner du temps.

6) Messages envoyés

Dans l’onglet Envoyés, vous voyez l’historique des messages que vous avez envoyés. Vous pouvez ouvrir un message pour relire le contenu.

7) Exemples d’utilisation (cas concrets)

Cellule Santé

Exemple: demander aux sections de remonter les informations santé.

  • Onglet Nouveau message
  • Filtre destinataires: Sections
  • Sélectionner toutes les sections
  • Envoyer la consigne

Cellule Corporatives

Exemple: informer les représentants de sections.

  • Onglet Nouveau message
  • Filtre destinataires: Sections
  • Sélectionner les sections concernées
  • Envoyer l’annonce

8) Exporter cette fiche en PDF

Sur Chrome / Edge (Windows):

  1. Ouvrir ce fichier: docs/fiche-messages.html
  2. Appuyer sur Ctrl + P
  3. Destination: Enregistrer au format PDF
  4. Format papier: A4 — Marges: Par défaut — Cocher “Arrière-plans” si nécessaire
  5. Cliquer sur Enregistrer
